Introduction

LOVE-aas (Love As A Service) provides a modern, RESTful, scalable solution to the common problem spreading the love.
Check out the examples!

How it works

Just type in your url bar any combination of the API keywords and then chose names
For example if Julie wants to send a crush message to John: just send him http://loveaas.com/crush/John/Julie !

API

All urls begin with http://loveaas.com
Use any of the available endpoints below:

/love/:to/:from

Hi :to, I love you. --:from

/xmas/:to/:from

Happy Christmas :to! --:from

/crush/:to/:from

I can't keep it in anymore, :to. I have a HUGE crush on you! --:from
