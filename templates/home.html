LOVE-AAS

Love As A Service

v0.1

Introduction

LOVE-aas (Love As A Service) provides a modern, RESTful, scalable solution to the common problem of telling people that you love them.

API

/love/:to/:from

Will return content in the form Hi :to, I love you. --:from

/miss/:to/:from

Will return content in the form Hi :to, I miss you, so much. --:from

/like/:to/:from

Will return content in the form Hi :to, I like you, like a lot. --:from
